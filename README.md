# Python + Basic Modules in ML

This folder contains **introductory Jupyter notebooks** for learning Python fundamentals and essential modules used in Machine Learning. It's ideal for beginners who want hands-on practice before diving into ML topics.

## What’s Included

### Essential Python Modules for ML
- `numpy` – Numerical computing and array operations  
- `pandas` – Data manipulation and analysis  
- `matplotlib` & `seaborn` – Data visualization  
- `scikit-learn` (`sklearn`) – Preprocessing, utilities, and basic ML workflows  

## Objectives
- Build a solid foundation in Python programming for ML  
- Learn to manipulate and explore datasets efficiently  
- Practice data visualization for analysis  
- Understand the role of Python modules in an ML workflow  

## How to Use
1. Open the notebooks in **Jupyter Notebook** or **VS Code**  
2. Follow the examples step by step  
3. Modify the code and experiment to reinforce learning
